# netplan配置双网卡	

系统: ubuntu-18.04.3-live-server-amd64

今天一台虚拟机要两个网卡，一个连接内网，一个连接外网，方便固定内网ip
## 修改 /etc/netplan/50-cloud-init.yaml

```sh 
vim /etc/netplan/50-cloud-init.yaml
```

```yml
network:
    network:
        ethernets:
            enp0s3:
                addresses:
                        - 192.168.56.20/24
            enp0s8:
                dhcp4: yes
        version: 2
```

## 使用
```sh 
netplan apply
```
